# Tiny Routine Log

A tiny TypeScript Node scaffold for tracking daily routines and streaks.

## Features
- Minimal TypeScript project structure
- Health entrypoint for quick checks
- Smoke test included
- Docker support

## Stack
- Node.js 20+
- TypeScript

## Setup
```bash
npm install
```

## Run
```bash
npm run dev
```

## Test
```bash
npm test
```

## Build
```bash
npm run build
npm start
```

## Docker
```bash
docker build -t tiny-routine-log .
docker run --rm -p 3000:3000 tiny-routine-log
```
