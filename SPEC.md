# Tiny Routine Log

## App Summary
Tiny Routine Log is a minimal TypeScript app for tracking daily routine completions and displaying streak stats in a lightweight API-friendly format.

## Target User
- Builders and operators who want a no-friction routine tracker.
- Developers needing a simple starter app that can be extended fast.

## MVP
1. Create routine items by name.
2. Mark routine completion by date.
3. Compute current streak + longest streak.
4. List routines with completion metadata.
5. Return health payload for smoke checks.

## Out of Scope (MVP)
- Multi-user auth
- Mobile UI
- Notifications
- Hosted database
